MacBook Air M4 系统恢复重置完整指南

⚠️ 重要提醒：
重置会删除所有数据，请先备份重要文件！

一、恢复前准备工作

  • 备份文件到 U 盘、移动硬盘或 iCloud
  • 退出 Apple ID（系统设置 → 顶部个人头像 → 退出登录）
  • 关闭 FileVault 加密（如已开启）
  • 连接电源，避免中途断电
  • 准备好 Wi‑Fi 密码（重装系统需要联网）

二、方案一：快速重置（推荐，保留当前系统）

适合：系统正常，只是文件/软件杂乱，想快速恢复干净状态。

  1. 点击左上角  → 系统设置
  2. 左侧选择 通用 → 传输或还原 Mac
  3. 点击 抹掉所有内容和设置
  4. 输入管理员密码，按提示确认抹掉
  5. 等待自动重启，进入初始化向导
  6. 选择不传输任何数据，即可得到全新系统

三、方案二：彻底重装系统（最干净）

适合：系统卡顿、异常、想完全清空磁盘重装。

1. 进入恢复模式

  1. 点击  → 关机
  2. 长按 电源键 不放，直到出现「加载启动选项」
  3. 选择 选项 → 继续
  4. 选择管理员账户，输入密码进入恢复工具

2. 抹掉硬盘

  1. 选择 磁盘工具 → 继续
  2. 左侧选中 APPLE SSD（顶层磁盘）
  3. 点右上角 抹掉
  4. 名称：Macintosh HD
    格式：APFS
    方案：GUID 分区图
  5. 确认抹掉，完成后退出磁盘工具

3. 重新安装 macOS

  1. 回到恢复界面，选择 重新安装 macOS
  2. 点继续，选择 Macintosh HD 作为目标磁盘
  3. 等待下载并安装（30~90 分钟，自动重启多次）
  4. 重启后按向导设置，选择不传输任何数据

四、初始化设置建议（保持干净）

  • 国家/地区：中国
  • 网络：连接 Wi‑Fi
  • 数据迁移：不传输任何信息
  • Apple ID：可先跳过，后续再登录
  • 创建本地账户（用户名+密码）
  • 关闭不必要的分析、定位等

五、常见问题

  • 抹不掉磁盘：关闭 FileVault 或退出 Apple ID
  • 下载慢：换 5GHz Wi‑Fi 或用手机热点
  • 无法进入恢复：关机 → 长按电源键直到出现选项
  • 激活锁：必须先退出原 Apple ID
✅ 恢复完成后建议：
只安装必要软件，关闭多余开机启动项，系统会非常清爽流畅。

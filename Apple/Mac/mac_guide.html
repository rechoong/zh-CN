🍎 Mac 新手入门学习笔记 ABC

从 Windows 20 年习惯到 macOS M4 的丝滑迁移指南

1. 核心思维与操作习惯

💡 核心差异： Mac 是“触控板优先” + “应用沙盒化”。不要试图把 Mac 变成 Windows，而是适应它的高效逻辑。

⌨️ 键位映射大转换

  • Ctrl (Win) → Command (⌘) (Mac)：复制/粘贴/撤销 (`⌘C/V/Z`)。
  • Alt (Win) → Option (⌥) (Mac)：辅助功能键。
  • Win (Win) → Command (⌘)：启动器是 `⌘ + Space` (Spotlight)。
  • 关闭窗口：左上角红点 (`⌘W` 关窗口，`⌘Q` 彻底退出程序)。
  • 强制退出：`⌘ + Option + Esc` (相当于 Ctrl+Alt+Del)。

🖱️ 触控板手势 (Trackpad)

  • 双指点击 = 右键菜单。
  • 三指拖移：需在“辅助功能”开启，实现不按压拖动窗口。
  • 四指上推：调度中心 (Mission Control)，查看所有窗口。
  • 四指左右滑：切换虚拟桌面 (强烈建议使用多桌面管理不同任务)。

2. 常用软件平替方案

🌐 网络环境

  • Clash Meta / Mihomo Party：主流订阅工具，界面友好。
  • Cloudflare WARP：官方客户端支持良好。
  • Surge / Loon：付费神器，适合开发者调试网络。

📥 下载工具 (IDM 平替)

  • Neat Download Manager：免费，最像 IDM，有浏览器嗅探插件。
  • Downie 4：强烈推荐。Mac 专属，支持全网视频下载，配合浏览器扩展一键搞定。
  • Motrix：开源免费，界面现代，支持 BT/磁力链。

💻 代码编辑器

  • HBuilder X：官方直接提供 Mac 版，无需替换。
  • Notepad++ 平替：推荐 Sublime Text (极速) 或 VS Code (生态最强)。
  • CotEditor：Mac 原生轻量编辑器，App Store 可下。

🐳 Docker 容器

  • Docker Desktop：官方标准方案。
  • OrbStack：强烈推荐。更轻、更快、更省内存，专为 Mac 优化。
  • 注意：M4 芯片为 ARM 架构，拉取镜像请优先选择 linux/arm64 版本。

🎬 视频剪辑与直播

  • 剪映：直接下载“剪映专业版”Mac 版，体验优于 Windows。
  • 专业剪辑：Final Cut Pro (买断，极致优化) 或 DaVinci Resolve (免费强大)。
  • OBS：有原生 Mac 版，运行稳定。
  • vMix 平替：Mac 无 vMix。推荐 Ecamm Live (功能强大但收费) 或使用 OBS 替代。

3. 一键视频转音频 (FFmpeg 脚本)

Windows 的 .bat 脚本在 Mac 上对应 .sh (Shell) 脚本。需先安装 Homebrew 和 FFmpeg。

安装前置：
1. 安装 Homebrew: /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
2. 安装 FFmpeg: brew install ffmpeg

将以下代码保存为 convert.sh，放入视频文件夹，终端运行 chmod +x convert.sh 然后 ./convert.sh。

#!/bin/zsh
# 脚本功能：批量将当前目录及子目录下的 mp4, ts, m4v 转换为 192k mp3

# 1. 检查 ffmpeg 是否已安装
if ! command -v ffmpeg > /dev/null; then
    echo "❌ ffmpeg 未找到，请先运行: brew install ffmpeg"
    exit 1
fi

echo "🚀 开始批量转换..."

# 2. 查找文件并循环处理
find . -type f \( -name "*.mp4" -o -name "*.ts" -o -name "*.m4v" \) -print0 | while IFS= read -r -d $'\0' file; do
    
    filename="${file%.*}"
    output="${filename}.mp3"
    
    if [ -f "$output" ]; then
        echo "⏭️  跳过已存在: $output"
        continue
    fi

    echo "🔄 转换中: $file"
    ffmpeg -y -i "$file" -f mp3 -ab 192k -vn "$output"
done

echo "✅ 全部完成！"

4. 避坑指南 & 必备工具箱

🛠️ 必装效率工具

  • Rectangle (免费)：窗口管理神器。让 Mac 拥有 Windows 一样的“拖拽边缘分屏”功能。
  • The Unarchiver (免费)：解压神器，支持 rar, 7z 等所有格式。
  • AppCleaner (免费)：卸载软件时，拖入图标可彻底删除残留配置文件。
  • IINA (免费)：基于 MPV 的本地播放器，颜值高，解码强。
  • Shottr：截图工具，支持长截图、打码、标注。

⚠️ 常见误区

  • 安装软件：下载 .dmg 文件后，必须把图标拖进“应用程序”文件夹才算安装完成！
  • 卸载软件：大多数软件直接把“应用程序”里的图标拖到废纸篓即可。
  • 移动硬盘：Mac 默认只能读 NTFS 不能写。建议硬盘格式化为 ExFAT 或安装 Mounty。

GitHub 从 0 到 1 全流程指南（MacBook Air M4）

适用于：Git 初学者、GitHub Pages、SSH 配置、分支管理、视频文件推送。

  2. 配置 Git 用户名与邮箱

  5. 创建 web 分支（用于网页端开发）

1. 检查 Git 是否已安装

git --version

MacBook Air M4 自带 Git，只要能看到版本号即可。

2. 配置 Git 用户名与邮箱

git config --global user.name "taoste"
git config --global user.email "taoste@gmail.com"

查看配置：

git config --list

3. 生成 SSH Key 并绑定 GitHub

3.1 生成 SSH Key

ssh-keygen -t rsa -b 4096 -C "taoste@gmail.com"

3.2 启动 SSH agent 并添加私钥

eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_rsa

3.3 查看公钥并复制

cat ~/.ssh/id_rsa.pub

3.4 添加到 GitHub

GitHub → Settings → SSH and GPG keys → New SSH Key

3.5 测试连接

ssh -T git@github.com

成功提示：

Hi taoste! You've successfully authenticated, but GitHub does not provide shell access.

4. 克隆仓库到本地

git clone git@github.com:ycdf/ycdf.github.io.git
cd ycdf.github.io

5. 创建 web 分支（用于网页端开发）

5.1 查看当前分支

git branch

5.2 创建并切换到 web 分支

git checkout -b web

5.3 推送 web 分支到 GitHub

git push -u origin web

6. 推送 41.5MB MP4 视频文件（可直接推送）

6.1 将视频放入仓库目录

cp /路径/sige.mp4 ~/ycdf.github.io/live/

6.2 添加并提交

git add .
git commit -m "添加司歌视频 sige.mp4（41.5MB）"

6.3 推送

git push

7. 解决 GitHub 邮箱隐私保护导致的推送失败（GH007）

错误提示：

remote: error: GH007: Your push would publish a private email address.

7.1 使用 GitHub 提供的匿名邮箱（推荐）

git config --global user.email "taoste@users.noreply.github.com"
git config user.email "taoste@users.noreply.github.com"

7.2 修复上一次提交的作者信息

git commit --amend --reset-author

7.3 强制推送

git push -f
